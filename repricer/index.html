Run Repricer (No Terminal)

Uses the same script/inputs as terminal: fetch SellCell feed, use /shc33/feed/feed.xml and /shc33/feed/amz.csv, write updates, and generate repricer-output.csv.

Pricing logic in use:
  • If profit is below target (default 15%), price is recalculated to hit target profit.
  • If profit is above target, bump amount is chosen by highest matching tier (15% / 45% / 75%).
  • Rounding: cents < 50 => .00, cents = 50 => .50, cents > 50 => next .00.
