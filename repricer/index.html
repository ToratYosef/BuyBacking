Run Repricer (No Terminal)

Runs the exact script flow as terminal: fetch SellCell feed, use /shc33/feed/feed.xml + /shc33/feed/amz.csv, then build preview output.

Pricing logic:
  • If profit is below target (default 15%), price is recalculated to hit target profit.
  • If profit is above target, bump amount is chosen by highest matching tier (15/45/75).
  • Rounding: < .50 => .00, = .50 => .50, > .50 => next .00.
