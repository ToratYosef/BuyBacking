Run Repricer (No Terminal)

This runs the same terminal script flow: fetch SellCell feed, use /shc33/feed/feed.xml and /shc33/feed/amz.csv, update prices, and generate preview CSV.

Pricing logic in use:

  • If profit is below target (default 15%), price is recalculated to hit the target profit.
  • If profit is above target, bump amount is chosen by the highest matching tier (15% / 45% / 75%).
